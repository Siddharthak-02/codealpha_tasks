# CodeAlpha - Student Grade Tracker (Java)

## Objective
A Java program to manage student grades and generate summary reports.

## Features
- Input grades for multiple students
- Calculates:
  - Average score
  - Highest score
  - Lowest score
- Stores grades using ArrayList
- Console-based interface

## How to Run
```bash
javac StudentGradeTracker.java
java StudentGradeTracker
